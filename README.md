#  DecodeLabs Java Project - Number Guessing Game

##  Overview
This is a simple console-based Number Guessing Game written in Java.  
The program generates a random number between 1 to 100, and the user will have 5 attempts to guess number correctly.

## Internship Details
This project was developed as part of a **Virtual Internship Program in Java Programming** at **DecodeLabs**.  
It was **Project 1**, which focus concepts like class, loops, conditionals, input handling.

---

## Features
- Random number generation (1–100)
- 5 chances to guess the number
- Feedback after each guess (Too High / Too Low / Correct)
- Input validation using exception handling
- Handles invalid inputs safely

---

## How It Works
- A random number is generated at the begining
- User enter their guesses
- Program gives feedback accordingly after each guess
- Shows remaining attempts
- Game ends when:
  - User guesses correctly OR
  - Attempts reach 5
